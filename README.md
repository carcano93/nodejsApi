# Nodejs Api

## Technologies:  

- express
- express-jwt
- passport
- routes
- mongoose
- jsonwebtoken

## How to run:

##### Build images: 
```
docker build -t nodeapi:latest .
```
##### Run containers: 
```
docker compose up -d
```
## Endpoints:

Get all endpoints info from: [http://localhost:3000/](http://localhost:3000/)
